Javascript Case Study

To Do List

할 일 데이터를 CRUD 방식으로 처리하였습니다.
기본 상태일때는 수정 및 삭제 버튼이 노출되고, 수정상태일때는 확인 및 취소 버튼이 활성화되도록 구현하였습니다.
Json-server를 활용해서 로컬에 REST API를 구축하고 연동하였습니다.

Infinite Scroll

컨텐츠의 끝이 브라우저 끝에 닿으면 api가 호출되어
새로운 데이터를 불러오도록 구현하였습니다. removeEventListener()로 계속 발생하는 이벤트를 명시적으로 해제하여 메모리 누수를 막았습니다. todolist와 달리 외부 api를 연동하였습니다.

3D Carousel

버튼은 클릭 시 rotateY()값이 60도씩 돌아가
화면이 움직이는 클릭이벤트를 구현하였습니다.
각 캐러셀에 index를 부여한 뒤 총 개수로 나누고 360도를 곱해 앵글의 위치값을 받아왔습니다.
버튼 클릭시 index가 변경되며 이벤트가 발생합니다.

Pop Up Modal

classList.toggle()를 활용하여 버튼 클릭 시
모달 창을 열고 닫는 이벤트를 구현하였습니다.
버튼이 아닌 모달 바깥 배경에도 클릭 이벤트를 적용해
바깥 배경을 눌러도 팝업이 닫히도록 구현하였습니다.
팝업을 띄울 시 바깥 스크롤 되지 않게 구현하였습니다.

Stop Watch

JS의 타이머 이벤트는 정확한 간격으로 발생하지 않기에
setInterval() 메서드를 사용해 경과된 시간을 구하고
Date.now()를 사용해 현재 시간을 가져와 현재 시간을 경과된 시간과 비교해 실제 경과 시간을 계산하는 방식으로 구현하였습니다.

Calculator

사칙연산, 소수점, 클리어, 초기화 기능을 구현하였습니다.
숫자 값을 저장하는 setNumber() 함수, 값을 초기화하는 reset() 함수 연산을 해주는 setOperation() 함수,
연산 후 값을 보여주는 finish() 함수 를 모든 계산기 패드에 클릭 이벤트로 구현해주었습니다.
